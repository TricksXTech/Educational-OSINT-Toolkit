# Educational OSINT Toolkit

A simple Python-based Open Source Intelligence (OSINT) toolkit for educational purposes.

This project demonstrates how publicly available information can be gathered using Python.

---

## Features

- Username search across social media
- Domain WHOIS lookup
- DNS/IP lookup
- Website metadata extraction
- Google Dork generator
- Interactive CLI menu

---

## Requirements

Install dependencies:

```bash
pip install requests beautifulsoup4 python-whois
```

Run Script
```bash
python osint.py
